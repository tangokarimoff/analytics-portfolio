# 👋 Артур Каримов — Data Analyst

Аналитик Данных, специализируюсь на построении аналитических решений полного цикла: от получения сырых данных (1С, Excel) до разработки ETL-пайплайнов, моделирования данных и создания BI-дашбордов в Yandex DataLens.

Работаю с SQL (включая оконные функции и оптимизацию запросов) - ClickHouse/PosgreSQL для обработки и трансформации данных.
Проектирую аналитические витрины данных и превращаю бизнес-задачи в решения, основанные на данных.

В настоящее время углубляюсь в Python и инженерию данных: оркестрация процессов, автоматизация и повышение надежности пайплайнов.
